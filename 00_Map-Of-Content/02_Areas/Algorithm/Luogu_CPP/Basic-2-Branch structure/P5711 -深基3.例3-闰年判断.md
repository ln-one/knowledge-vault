## Description

输入年份，若能被400整除或能被4整除但不能被100整除，则输出1，否则输出0。
### Example:  
Input: 
```json
1926
```

Output: 
```json
0
```
## Question Link

[洛谷 - P5711【深基3.例3】闰年判断](https://www.luogu.com.cn/problem/P5711)
## Approach

- 非常简单，如果`x % 400 = 0` or `x % 4 = 0` and `x % 100 != 0`。然后结束了

## Implementation

```python
x = int(input())
if x % 400 == 0 or x % 4 == 0 and x % 100 != 0:
    print(1)
else:
    print(0)
```
