## Description
一些整数可能拥有以下的性质：
- 性质 1：是偶数；
- 性质 2：大于 4 且不大于 12。
小 A 喜欢这两个性质同时成立的整数；Uim 喜欢这至少符合其中一种性质的整数；小 B 喜欢刚好有符合其中一个性质的整数；正妹喜欢不符合这两个性质的整数。现在给出一个整数 x，请问他们是否喜欢这个整数？
### Example:  
Input: 
```json
12
```

Output: 
```json
1 1 0 0
```
## Question Link

[P5710 【深基3.例2】数的性质 - 洛谷](https://www.luogu.com.cn/problem/P5710)
## Approach

- 性质A:偶数：`x%2==0`
- 性质B:大于4且不大于12。
## Implementation

```python
num = int(input())
if num % 2 == 0:
    A = 1
else:
    A = 0
if num > 4 and num <= 12:
    B = 1
else:
    B = 0

print(f"{A and B} {A or B} {int(A and not B or not A and B)} {int(not A and not B)}")
```

## Analysis
- 需要注意的是：`not`运算会把`0->Ture` 而`1->False`，所以最后输出的最后一项是布尔类型而不是0/1，所以要使用`int()`进行转换。
