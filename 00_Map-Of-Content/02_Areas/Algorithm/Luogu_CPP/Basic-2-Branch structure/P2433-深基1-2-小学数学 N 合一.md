elif a == 14:
    # 收入 3500 元，课程定价与报名人数线性关系

    import math

    a_coef = 1
    b_coef = -120

    delta = b_coef**2 - 4*a_coef*c_coef

## Analysis
### （1）`if` 语句
- 最简单的分支结构
- 条件成立则执行对应代码块，否则跳过

```Python
x = 10

if x > 5:
    print("x 大于 5")
```

- 注意 Python 使用 **冒号 `:`** 和 **缩进** 来区分代码块

---
### （2）`if-else` 语句
- 条件不成立时执行另一块代码
```Python
x = 3

if x > 5:
    print("x 大于 5")
else:
    print("x 小于等于 5")

```
---
### （3）`if-elif-else` 语句
- 多分支选择，类似 C++ 的 `if - else if - else`
```Python
x = 7

if x < 5:
    print("小于 5")
elif x == 5:
    print("等于 5")
elif x < 10:
    print("大于 5 且小于 10")
else:
    print("大于等于 10")

```
- Python 的 `elif` 相当于 C++ 的 `else if`
- 最后的 `else` 可以省略
---
### （4）Python 3.10+ 的 `match-case`（类似 C++ `switch-case`）
- 语法上最接近 C++ 的 `switch-case`
- 支持多值匹配、解构等
```Python
value = 2

match value:
    case 1:
        print("这是 1")
    case 2 | 3:
        print("这是 2 或 3")
    case _:
        print("其他情况")

```
- `case _` 相当于 `default`
- 可以匹配多个值：`2 | 3`
- 可以匹配复杂对象模式（列表、字典等）
### (5) `ord()` 是 Python 的一个内置函数，用来 **返回一个字符对应的 Unicode（或者 ASCII）整数编码**。
