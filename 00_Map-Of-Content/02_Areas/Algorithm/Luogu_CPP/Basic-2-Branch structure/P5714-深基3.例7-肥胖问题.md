## Description

根据体重 m（kg）和身高 h（m）计算 BMI = $\frac{m}{h^2}$，再根据范围输出体型：

- BMI < 18.5 → **Underweight**
    
- 18.5 ≤ BMI < 24 → **Normal**
    
- BMI ≥ 24 → 输出 BMI（保留六位有效数字，去掉多余 0），再输出 **Overweight**。
### Example:  
Input: 
```json
70 1.72
```

Output: 
```json
Normal
```
## Question Link

[洛谷 - P5714 【深基3.例7】肥胖问题](https://www.luogu.com.cn/problem/P5714)
## Approach

- 关键在于`.6g`代表的是六位有效数字[[P5706【深基2.例8】再分肥宅水#**Python f-string 知识点总结**]]


## Implementation

```python
m, h = map(float, input().split())
BMI = m / (h * h)
if BMI < 18.5:
    print("Underweight")
elif BMI >= 24:
    print(f"{BMI:.6g}")
    print("Overweight")
else:
    print("Normal")
```
