## Description

小 B 喜欢吃苹果。她现在有 m（1≤m≤100）个苹果，吃完一个苹果需要花费 t（0≤t≤100）分钟，吃完一个后立刻开始吃下一个。现在时间过去了 s（1≤s≤10000）分钟，请问她还有几个完整的苹果？
### Example:  
Input: 
```json
50 10 200
```

Output: 
```json
30
```
## Question Link

[P5709 【深基2.习6】Apples Prologue / 苹果和虫子 - 洛谷](https://www.luogu.com.cn/problem/P5709)
## Approach

- 用`s//t`作为可以吃掉多少个苹果，其中如果`t=0`，那么直接返回0。
	- 然后计算`s%t`，如果剩下的时间不为0，则再令可以吃掉的苹果数量+1。
- 接下来判断能够吃掉的苹果数量是否会大于有的苹果数。
	- 如果大于，则返回0
	- 如果不大于，则返回原有的苹果数量-吃掉的苹果数


## Implementation

```python
m, t, s = map(int, input().split())
if t == 0:
    print(0)
else:
    if (s % t) == 0:
        eatedNum = s // t
    else:
        eatedNum = s // t + 1
    if eatedNum >= m:
        print(0)
    else:
        print(m - eatedNum)

```
