## Description
**Today, I ate x apple(s).** ——当 x ≠ 1 时在 apple 后加 **s**，当 x=0 时输出 0。
### Example:  
Input: 
```json
1
```

Output: 
```json
Today, I ate 1 apple.
```
## Question Link

[洛谷 - P5712 【深基3.例4】Apples](https://www.luogu.com.cn/problem/P5712)
## Approach
- 如果输入x大于1，则改为apples。
## Implementation

```python
x = int(input())
if x > 1:
    print(f"Today, I ate {x} apples.")
else:
    print(f"Today, I ate {x} apple.")
```
