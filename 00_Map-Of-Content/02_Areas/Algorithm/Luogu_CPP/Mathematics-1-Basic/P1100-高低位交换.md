## Description

将一个 32 位整数的高 16 位与低 16 位交换后输出新的十进制值。
### Example:  
Input: 
```
1314520
```

Output: 
```
249036820
```
## Question Link

[洛谷-P1100-高低位交换](https://www.luogu.com.cn/problem/P1100)
## Approach

- 首先因为题目上说是**非负整数并且为32位**，则我们确定必须使用==`unsigned int`==来进行操作，需要注意的是==C语言中是没有指定逻辑右移==的符号的，例如Java中的`>>>`,所以我们如果想要进行逻辑右移，则必须使用`unsigned`，或者使用掩码[[Data-lab#4.allOddBits]] .
- 接下来定义`low_16`代表的是*变换后的结果*低16位，高位赋0，`high_16`也是同样，是高16位。
- 然后我们直接用`或（|）`来将==`low_16`和`high_16`组合==形成最终的结果`res`，这样就完成了这道题。

## Implementation

```cpp
#include <iostream>

using namespace std;

unsigned int n; // 因为题目上说是非负整数，并且是32位的，所以我们这里用unsigned int

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  cin >> n;
  unsigned int low_16 = n >> 16;
  unsigned int high_16 = n << 16;
  unsigned int res = low_16 | high_16;
  cout << res << endl;
  return 0;
}
```
