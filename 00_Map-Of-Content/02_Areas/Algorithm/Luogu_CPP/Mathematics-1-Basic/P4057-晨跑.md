## Description

求三个数的最小公倍数
### Example:  
Input: 
```
2 3 5
```

Output: 
```
30
```
## Question Link

[P4057 [Code+#1] 晨跑 - 洛谷](https://www.luogu.com.cn/problem/P4057)
## Approach

- 题目说的花里胡哨，其实就是求三个数的最小公倍数。
- 首先考虑到我可以先求前两个数的最小公倍数，然后再将这个最小公倍数和第三个数求出最后的答案。
- 那么如何求两个数的最小公倍数`lcm`呢？[[P1572-计算分数#1.最小公倍数公式]]


## Implementation

```cpp
#include <iostream>

using namespace std;

typedef long long ll;

ll lcm(ll x, ll y) {
  ll origin_x = x;
  ll origin_y = y;
  while (y != 0) {
    ll t = x % y;
    x = y;
    y = t;
  }
  return origin_x * origin_y / x;
}

int main() {
  ll a, b, c;
  cin >> a >> b >> c;
  ll temp = lcm(a, b);
  ll ans = lcm(temp, c);
  cout << ans << endl;
  return 0;
}

```

## Analysis
- 很简单的一道题，需要记的就是`lcm`的公式了。
