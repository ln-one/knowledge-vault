## Description

求从 1 到 n 的所有正整数的约数个数之和。
### Example:  
Input: 
```
3
```

Output: 
```
5
```
## Question Link

[P1403 [AHOI2005] 约数研究 - 洛谷](https://www.luogu.com.cn/problem/P1403)
## Approach

- 如果一个一个找，会很慢，所以我们逆向思维，思考：每个i会成为哪些数的约数？
- 答案是每个i会成为`n / i`个数的约数。
- 因此我们只需要把从1到n的所有约数加起来就可以了。

## Implementation

```cpp
#include <iostream>

using namespace std;

int n;

int main() {
  cin >> n;
  long long ans = 0;
  for (int i = 1; i <= n; i++) {
    ans += n / i;
  }
  cout << ans << endl;
}


```

## Analysis
我们反向思考：每个数 `i` 会成为哪些数的约数？  
答案是：`i` 是 `n / i` 个数的约数。

因此我们可以遍历 `i = 1` 到 `n`，将每个 `n / i` 累加起来：

```cpp
long long ans = 0;
for (int i = 1; i <= n; ++i) {
    ans += n / i;
}
