## Description

在一个长方形田地中，用尽量少的体力（每种一个正方形耗费周长）填满整个田地，求最小体力值。
### Example:  
Input: 
```
1 10
```

Output: 
```
40
```
## Question Link

[P2660-种田-洛谷](https://www.luogu.com.cn/problem/P2660)
## Approach

1. **贪心切最大正方形**：每次都种能种的最大正方形（边长为较小边 y），就能保证体力最少。
2. **GCD 关联**：每次 x %= y 就像欧几里得求 GCD，最终矩形会被完全分割。
3. **体力计算**：每个正方形体力 = 周长 = 4 * 边长，乘以数量 x / y。


## Implementation

```cpp
#include <iostream>

using namespace std;

int main() {
  long long x, y, ans = 0;
  cin >> x >> y;
  while (y && x) {
    if (x < y)
      swap(x, y);
    ans += 4 * y * (x / y);
    x %= y;
  }

  cout << ans << endl;
  return 0;
}

```
