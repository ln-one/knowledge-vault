## Description

输入`a`和`b`，输出`a * b`
### Example:  
Input: 
```json
3 5
```

Output: 
```json
15
```
## Question Link

[洛谷-P5703【深基2.例5】苹果采购](https://www.luogu.com.cn/problem/P5703)
## Approach

- 使用`map`函数将输入先转换为整数存入`a`和`b`中，然后输出即可

## Implementation

```python
a,b = map(int,input().split())
print(a*b)
```
