## Description
输入一个不小于 100 且小于 1000，同时包括小数点后一位的一个浮点数，例如 123.4 ，要求把这个数字翻转过来，变成 4.321 并输出。
### Example:  
Input: 
```json
123.4
```

Output: 
```json
4.321
```
## Question Link

[# P5705 【深基2.例7】数字反转- 洛谷](https://www.luogu.com.cn/problem/P5705)
## Approach

- 使用slice[]分片


## Implementation

```python
num = input()
print(num[::-1])
```

## Analysis
### 切片(slice)：
### **一、基本语法**

```python
sequence[start:end:step]
````
> 注意是[],而不是()


适用于：

- 字符串（str）
- 列表（list）
- 元组（tuple）

|**参数**|**含义**|
|---|---|
|start|起始索引（包含）|
|end|结束索引（不包含）|
|step|步长，控制取值方向与间隔|

索引从 **0 开始**，负数表示从右往左（如 -1 表示最后一个元素）。

---

### **二、基本示例**

```
s = "abcdef"

s[0:3]   # → 'abc'   从索引 0 到 2
s[2:5]   # → 'cde'   从索引 2 到 4
s[:3]    # → 'abc'   省略 start，默认从开头开始
s[3:]    # → 'def'   省略 end，默认到结尾
s[:]     # → 'abcdef' 全部复制
```

---

### **三、步长（step）**

```
s = "abcdef"

s[::2]   # → 'ace'   每隔一个取一个
s[::3]   # → 'ad'    每隔两个取一个
```

当 step 为负数时，表示**反向取值**。

---

### **四、反向切片（重点）**

```
s = "abcdef"

s[::-1]  # → 'fedcba'   反转整个字符串
s[::-2]  # → 'fdb'      每隔一个反向取
```

#### **解析：**

|**参数**|**含义**|
|---|---|
|start|省略 → 默认从结尾开始|
|end|省略 → 默认到开头结束|
|step = -1|每次向左取一个字符|

---
### **五、负索引**

```
s = "abcdef"

s[-3:]   # → 'def'   从倒数第3个开始取到结尾
s[:-2]   # → 'abcd'  从开头取到倒数第3个前一个
```
