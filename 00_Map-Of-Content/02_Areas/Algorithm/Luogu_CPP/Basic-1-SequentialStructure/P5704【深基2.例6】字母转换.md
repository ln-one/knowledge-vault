## Description

输入一个小写字母，输出其对应的大写字母。例如输入 q时，会输出 Q。
### Example:  
Input: 
```json
q
```

Output: 
```json
Q
```
## Question Link

[洛谷-P5704-字母转换](https://www.luogu.com.cn/problem/P5704)
## Approach

- upper()


## Implementation

```python
a = input()
print(a.upper())
```
