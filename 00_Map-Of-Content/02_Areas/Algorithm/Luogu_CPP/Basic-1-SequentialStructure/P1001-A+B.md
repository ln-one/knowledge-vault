## Description

输入两个整数 a,b，输出它们的和
### Example:  
Input: 
```json
20 30
```

Output: 
```json
50
```
## Question Link

[Luogu - Two Sum](https://www.luogu.com.cn/problem/P1001)
## Approach

# Python 输入与处理常用函数

1. **input()**
   - 用途：从控制台读取一行文本（返回字符串类型）。
   - 示例：
     ```python
     s = input("请输入内容: ")
     print(s)
     ```

2. **split()**
   - 用途：将字符串按指定分隔符拆分成列表，默认按空格。
   - 示例：
     ```python
     s = "10 20 30"
     lst = s.split()  # ['10', '20', '30']
     lst2 = s.split(',')  # 按逗号拆分
     ```

3. **map()**
   - 用途：对可迭代对象中的每个元素应用指定函数，返回 map 对象（可转 list）。
   - 示例：
     ```python
     lst = ['1', '2', '3']
     nums = list(map(int, lst))  # [1, 2, 3]
     ```

4. **结合使用（常见用法）**
   - 从一行输入中读取多个整数：
     ```python
     a, b = map(int, input("请输入两个数字，用空格分隔: ").split())
     print(a + b)
     ```
   - 解释：
     1. `input()` 读取一行字符串 `"10 20"`  
     2. `.split()` 拆分成列表 `['10', '20']`  
     3. `map(int, ...)` 将列表中的每个元素转换为整数  
     4. `a, b = ...` 直接解包成两个变量

## Implementation

```python
a,b = map(int,input().split())
print(a+b)
```
