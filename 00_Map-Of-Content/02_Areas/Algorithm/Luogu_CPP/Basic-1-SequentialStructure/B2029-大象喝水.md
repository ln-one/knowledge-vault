## Description
一只大象口渴了，要喝 20 升水才能解渴，但现在只有一个深 h 厘米，底面半径为 r 厘米的小圆桶 （h和 r 都是整数）。问大象至少要喝多少桶水才会解渴。
### Example:  
Input: 
```json
23 11
```

Output: 
```json
3
```
## Question Link

[洛谷 - B2029 大象喝水](https://www.luogu.com.cn/problem/B2029)

## Implementation

```python
h,r = map(int,input().split())
volume = r * r * 3.14 * h / 1000
print(int(20 // volume + 1))
```
