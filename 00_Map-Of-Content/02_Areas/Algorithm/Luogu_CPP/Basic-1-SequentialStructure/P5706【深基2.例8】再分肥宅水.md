## Description
现在有 t 毫升肥宅快乐水，要均分给 n 名同学。每名同学需要 2 个杯子。现在想知道每名同学可以获得多少毫升饮料（严格精确到小数点后 3 位），以及一共需要多少个杯子。
### Example:  
Input: 
```json
500.0 3
```

Output: 
```json
166.667
6
```
## Question Link

[洛谷-P5706 【深基2.例8】再分肥宅水](https://www.luogu.com.cn/problem/P5706)
## Approach

- 关键问题是要求输出的是固定位数的小数


## Implementation

```python
t,n = map(float,input().split())
print(f"{t/n:.3f}")
print(n*2)
```

## Analysis
## **Python f-string 知识点总结** 
### **一、基本语法**

```
name = "Evan"
print(f"Hello, {name}!")
```

输出：

```
Hello, Evan!
```

**语法说明：**

- 在字符串前加 f 或 F
- 用 {} 包裹变量或表达式
- Python 会自动将花括号中的内容求值并插入字符串中
---

### **二、支持表达式**

```
a, b = 3, 4
print(f"{a} + {b} = {a + b}")
```

输出：

```
3 + 4 = 7
```

你可以在 {} 里放任何合法表达式（包括函数调用）。

---
### **三、控制小数位数**
```
num = 3.1415926
print(f"{num:.2f}")   # 保留2位小数
print(f"{num:.4f}")   # 保留4位小数
```
输出：
```
3.14
3.1416
```
常见格式控制：

|**格式**|**含义**|**示例**|**输出**|
|---|---|---|---|
|{x:.2f}|保留2位小数|3.14159|3.14|
|{x:.0f}|四舍五入为整数|3.6|4|
|{x:8.2f}|宽度8位，保留2位小数|3.14|'    3.14'|
|{x:08.2f}|宽度8位，不足补0|3.14|'00003.14'|

---
### **四、对齐与填充**
```
text = "AI"
print(f"{text:<10}")   # 左对齐
print(f"{text:>10}")   # 右对齐
print(f"{text:^10}")   # 居中
print(f"{text:*^10}")  # 居中，用*填充
```
输出：
```
AI        
        AI
    AI    
****AI****
```
---
### **五、整数格式控制**
```
n = 42
print(f"{n:04d}")    # 不足4位补0
print(f"{n:+d}")     # 显示正负号
print(f"{n:,}")      # 加千位分隔符
```
输出：
```
0042
+42
42
```
---
### **六、不同进制输出**
```
n = 255
print(f"{n:b}")  # 二进制
print(f"{n:o}")  # 八进制
print(f"{n:x}")  # 十六进制小写
print(f"{n:X}")  # 十六进制大写
```
输出：
```
11111111
377
ff
FF
```
---
### **七、字符串格式化**
```
word = "Python"
print(f"{word:.3}")   # 截取前3个字符
print(f"{word:>10}")  # 右对齐
```
输出：
```
Pyt
    Python
```
---
### **八、嵌套表达式**
```
x = 10
print(f"{x * 2 + 1}")         # 表达式
print(f"{x / 3:.2f}")         # 表达式 + 格式控制
```
输出：
```
21
3.33
```
---
### **九、在花括号中显示大括号**
```
print(f"{{Hello}}")  # 输出大括号
```
输出：
```
{Hello}
```
