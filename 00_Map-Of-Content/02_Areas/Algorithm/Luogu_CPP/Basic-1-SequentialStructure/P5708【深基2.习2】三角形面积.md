## Description

一个三角形的三边长分别是 a、b、c，那么它的面积为 $\sqrt{p(p−a)(p−b)(p−c)​}$，其中 p=$\frac{1}{2}$(a+b+c)。输入这三个数字，计算三角形的面积，四舍五入精确到 1 位小数。
### Example:  
Input: 
```json
3 4 5
```

Output: 
```json
6.0
```
## Question Link

[洛谷 - P5708 【深基2.习2】三角形面积](https://www.luogu.com.cn/problem/P5708)
## Approach

- 使用幂运算符`**`来进行运算，开方就是对$\frac{1}{2}$进行幂运算。


## Implementation

```python
a,b,c=map(float,input().split())
p=(a+b+c)/2
res = float((p*(p-a)*(p-b)*(p-c))**0.5)
print(f"{res:.1f}")
```

## Analysis
使用幂运算符`**`来进行运算，开方就是对$\frac{1}{2}$进行幂运算。
