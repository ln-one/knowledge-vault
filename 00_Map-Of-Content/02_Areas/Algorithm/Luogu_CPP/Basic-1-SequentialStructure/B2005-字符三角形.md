## Description

给定一个字符，用它构造一个底边长 5 个字符，高 3 个字符的等腰字符三角形。
### Example:  
Input: 
```json
*
```

Output: 
```json
  *
 ***
*****
```
## Question Link

[洛谷 - B2005-字符三角形](https://www.luogu.com.cn/problem/B2005)
## Approach

1. `* `可以用来重复字符或字符串，例如 `' ' * 3 → '   '`
2. print() 会自动换行
3. `range(1, h+1)`等价于 C++ 的 `for(int i=1;i<=h;i++)`


## Implementation

```python
sign = input()
h = 3

for i in range(1,h+1):
    print(' ' * (h - i) + sign * (2*i-1))

```

## Analysis
1. **for 循环**
   - 功能：重复执行一段代码
   - Python 写法：
     ```python

     for i in range(1, h+1):  # i 从 1 到 h

         print(i)

     ```
   - 对比 C++：
     ```cpp

     for(int i=1;i<=h;i++) { ... }

     ```
2. **range()**
   - 生成数字序列，可用于循环
   - 示例：
     ```python

     range(1, h+1)  # 生成 1,2,3

     range(5)       # 生成 0,1,2,3,4

     ```
3. **字符串重复**
   - 用 `*` 来重复字符或字符串
   - 示例：
     ```python

     print(' ' * 3)  # 打印三个空格

     print('*' * 5)  # 打印 *****

     ```
4. **字符串拼接**
   - 用 `+` 把多个字符串连接起来
   - 示例：
     ```python

     print(' ' * (h-i) + c * (2*i-1))
