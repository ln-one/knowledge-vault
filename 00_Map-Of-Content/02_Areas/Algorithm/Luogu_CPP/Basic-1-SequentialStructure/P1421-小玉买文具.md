## Description

班主任给小玉一个任务，到文具店里买尽量多的签字笔。已知一只签字笔的价格是 1 元 9 角，而班主任给小玉的钱是 a 元 b 角，小玉想知道，她最多能买多少只签字笔呢。
### Example:  
Input: 
```json
10 3
```

Output: 
```json
5
```
## Question Link

([P1421 小玉买文具 - 洛谷](https://www.luogu.com.cn/problem/P1421))
## Approach

- 将元全部转换成给的钱的角数//(整除)签字笔的角数即可。  

## Implementation

```python
a,b = map(int,input().split())
b += a*10
print(b//19)
```

## Analysis
- 需要 `/`是浮点数除法，我们要求的是整数，所以必须使用`//`。
