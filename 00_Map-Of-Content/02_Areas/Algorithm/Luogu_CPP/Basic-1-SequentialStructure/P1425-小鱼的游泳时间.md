## Description

### Example:  
Input: 
```json
12 50 19 10
```

Output: 
```json
6 20
```
## Question Link

[洛谷 - P1425-小鱼的游泳时间](https://www.luogu.com.cn/problem/P1425)
## Approach

- 如果d小于b，则让d+60，并让a减去1，然后计算c-a
## Implementation

```python
a, b, c, d = map(int, input().split())
if d < b:
    d = d + 60 - b
    c = c - 1 - a
else:
    d = d - b
    c = c - a
print(f"{c} {d}")
```
