## Description

给定两整数 **s**（路程，单位米）和 **v**（速度，单位米/分钟），表示 yyy 从家到学校需要走 s / v 分钟。另外，yyy 在途中还要 **额外花 10 分钟** 进行垃圾分类。学校要求 **必须在 8:00（上午八点）之前到达**。要求计算：yyy **最晚可以出门的时间**（不迟到为准）。输出格式为 **24小时制的 HH:MM**，必须补足两位（例如 07:05）。若出发时间早于前一天的 00:00，则要从前一天倒推（比如 23:50）。
### Example:  
Input: 
```json
100 99
```

Output: 
```json
07:48
```
## Question Link

[洛谷 - P5707 【深基2.例12】上学迟到](https://www.luogu.com.cn/problem/P5707)
## Approach

- 路程除以速度，然后向上取整。
- 然后总时间为`24 * 60 = 1440`
- 用总时间减去消耗的时间
- 然后再转换成实际的时间
- 如果实际的时间小于960,则时间为从8点加上小时，分钟
- 如果大于960,则先减去960,然后转换为小时，分钟


## Implementation

```python
s,v=map(int,input().split())
time = 60 * 24 - (s/v) - 10
if time < 960:
    hour = int(8 + time // 60)
else:
    hour = int((time-960) // 60)
minute = int(time % 60)

print(f"{hour:02d}:{minute:02d}")
```

## Analysis
- python除号默认是浮点数除法，要想整除(向下整除，即结果不大于实际结果)，则需要使用`//`
