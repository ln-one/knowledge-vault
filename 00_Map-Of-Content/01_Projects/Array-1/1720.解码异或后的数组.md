## Description
**未知** 整数数组 `arr` 由 `n` 个非负整数组成。

经编码后变为长度为 `n - 1` 的另一个整数数组 `encoded` ，其中 `encoded[i] = arr[i] XOR arr[i + 1]` 。例如，`arr = [1,0,2,1]` 经编码后得到 `encoded = [1,2,3]` 。

给你编码后的数组 `encoded` 和原数组 `arr` 的第一个元素 `first`（`arr[0]`）。

请解码返回原数组 `arr` 。可以证明答案存在并且是唯一的。
### Example:  
Input: 
```json
encoded = [1,2,3], first = 1
```

Output: 
```json
[1,0,2,1]
```
## Question Link

[LeetCode](https://leetcode.cn/problems/decode-xored-array/)
## Approach

- 已知：encoded[i] = arr[i] ^ arr[i+1]
- 通过异或性质：x ^ y ^ y = x，可得到
    arr[i+1] = encoded[i] ^ arr[i]
- 从已知的 arr[0] = first 依次推算即可得到整个原数组。

## Implementation

```python
class Solution:
    def decode(self, encoded: List[int], first: int) -> List[int]:
        # 解码后的原数组 arr，其中第一个元素已知
        arr = [first]

        # 利用关系：encoded[i] = arr[i] XOR arr[i+1]
        # 可推出：arr[i+1] = encoded[i] XOR arr[i]
        for e in encoded:
            arr.append(arr[-1] ^ e)   # 异或还原下一个元素

        return arr
```

## Analysis
- 关键在于`异或性质：x ^ y ^ y = x`
