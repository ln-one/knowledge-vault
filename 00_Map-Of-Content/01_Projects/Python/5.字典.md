## 1.使用字典：
### 1.1字典：
- 用`{}`包起来。
- 字典是一系列键值对，每个键都与一个值相关联。
```Python
alien_0 = {'color':'green','points':5}
```
### 1.2访问字典中的值：
```Python
print(aline_0['color'])
```
### 1.3添加键值对：
```Pyhon
alien_0['x_position'] = 0
alien_0['y_position'] = 25
```
- 字典是一种动态结构，可随时在其中添加键值对
### 1.4.先创建一个空字典：
```Python
alien_0 = {}
```
### 1.5.修改字典的值：
```Python
alien_0 = {'color':'green'}
alien_0['color'] = 'yellow'
```
### 1.6.删除键值对：
使用`del`语句将相对应的键值彻底删除。使用del必须指定字典名和 要删除的键。
```Python
del alien_0['points']
```
### 1.7.使用get()来访问值：
```Python
print(alien_0['points'])
```
如果要获取外星人的分数，结果会是：Python显示traceback，指出存在键值错误(KeyError)。

方法`get()`的第一个参数用于指定键，第二个参数为指定的键不存在时要返回的值 ，是可选的。
```Python
point_value = alien_0.get('point','No point value assigned.')
```
## 2.遍历字典
### 2.1.遍历所以键对值：
```Python
for key,value in user_0.items():
	print(f"\nKey:{key}")
	print(f"Value:{value}")
```
### 2.2.遍历字典中的所有键：
```Python
for name in favorite_languages.key():
	print(name.title())
```
遍历字典时，会默认遍历所有的键。
```Python
for name in favorite_languages:
	print(name.title())
```
同上面的一致。
- .key()实际上返回一个列表，其中包含了所有的键的值。
### 2.3.按特定顺序遍历字典中的所有键：
```Python
for name in sorted(favorite_languages.key())
	print(f"{name.title()}")
```
### 2.4.遍历字典中的所有值：
```Python
for language in favorite_languages.value():
	print(language.title())
```
- 可以用set()来进行去重：
```Python
for language in set(favorite_languages.value())
	print(language.title())
```
## 3.嵌套：
- 可以将字典嵌入进列表中：
```Python
aliens = [alien_0,alien_1,alien]
for alien in aliens:
	print(alien)
```
- 可以将列表嵌入到字典中：
- ```Python
  pizza = {
	  'curst' : 'thick',
	  'toppings': ['mushroom','extra cheese'],
  }
  for topping in pizza['toppings']:
	  print(topping)
  ```
  - 可以将字典嵌入到字典中.
