## 1.字符串
### 1.1.使用方法修改字符串的大小写：
```python
name = "ada lovelace"
print(name.title())
print(name.upper())
print(name.lower())
```
Output:
```
Ada lovelace
ADA LOVELACE
ada lovelace
```
### 1.2.在字符串中使用变量：
```python
first_name = "ada"
last_name = "lovelace"
full_name = f"{first_name} {last_name}"
print(full_name)
```
Output:
`ada lovelace`
### 1.3.使用制表符或换行符来添加空白
- 制表符：`\t`
- 换行符：`\n`
### 1.4.删除空白
- 删除右边的空白：`rstrip()`
- 删除左边的空白：`lstrip()`
- 删除左右两边的空白：`strip()`
```python
favorite_language = 'python '
favorite_language.rstrip()
'python'
```
## 2.数
### 2.1.整数：
`**`可以直接表示为乘方运算
### 2.2.整数和浮点数：
- 当任意两个数相除时，结果总是浮点数，即便这两个数都是整数且能整除
- 无论是哪种运算，只要操作数是浮点数，Python默认得到的总是浮点数，即便结果原本是整数也是如此。
### 2.3.数中的下划线：
书写很大的数时，可使用下划线将其中的数字分组，使其更清晰易读
`universe_age = 14_000_000_000`
在`print`时，不会出现_
### 2.4.同时给多个变量赋值：
可以同时给多个变量赋值：
`x,y,z = 0 , 0 , 0`
### 2.5.常量
Python中没有内置的常量类型，但Python程序员通常会使用全大写来指出应将某个变量视为常量
