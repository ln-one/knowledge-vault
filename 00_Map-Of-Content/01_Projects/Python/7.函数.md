## 1.定义函数
```Python
def greet_user():
	print("Hello!")
	
greet_user()
```
### 1.1.向函数传递信息：
```Python
def greet_user(username):
	print(f"Hello,{username.title()}")

greet_user('evan')
```
- 定义的虚值占位符username：**形参**(parameter)
- 实际的数值evan：**实参**(argument)
## 2.传递参数
### 2.1.位置传参：

def describe_pet(animal_type,pet_name):

## 5.传递任意数量的参数：
Python允许函数从调用语句中收集任意数量的实参。
```Python
def make_pizza(*toppings):
	print(toppings)
	
make_pizza('peperoni')
make_pizza('mushroom','green peppers','extra cheese')
```
### 5.1.结合使用位置实参和任意数量实参：
如果要让函数接受不同类型的实参，必须在函数定义中将接纳任意数量实参的形参放在最后。
```Python
def make_pizza(size,*toppings):
	print(f"\nMaking a {size}-inch pizza with the following toppings:")
	for topping in toppings:
		print(f"-{topping}")
```
### 5.2.使用任意数量的关键字实参：
```Python
def build_profile(first,last,**user_info):
	user_info['first_name'] = first
	user_info['last_name']= last
	return user_info

user_profile = build_profile('albert','einstein',location='princeton',field='physics')
print(user_profile)
```
## 6.将函数存储在模块中
### 6.1.导入整个模块：
**模块**是扩展名为.py的文件。
导入模块：
`import pizza`
### 6.2.导入特定的函数：
导入模块中的特定函数：
```Python
from module_name import function_name

#导入任意数量的函数：
from module_name import funciton_0,function_1,function_2

#指定别名：
from pizza import make_pizza as mp

#使用as给模块指定别名
import pizza as p

#导入模块中的所有函数
from pizza import *

```
import语句中的星号让Python将模块中的每个函数复制到这个程序文件中。由于导入了每个函数，可通过名称来调用每个函数，而无须使用句点表示法。
- 函数命名一般是通过**小写字母**加**下划线**，Python程序员们偏爱这个，而非驼峰命名法。
