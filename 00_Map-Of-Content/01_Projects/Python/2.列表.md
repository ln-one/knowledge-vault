```python
>>> bicycles = ['trek','cannodale','redline','specialized']
>>> print(bicycles)
['trek', 'cannodale', 'redline', 'specialized']
```
## 1.修改、添加和删除元素
### 1.1.修改列表元素
```python
bicycles[0] = 'email'
print(bicycles[0])
email
```
### 1.2.在列表中添加元素
#### 1.2.1.在列表末尾添加元素：
```python
bicycles.append('ducati')
```
#### 1.2.2.在列表中插入元素：
```python
bicycles.insert(0,'ducati')
```
`insert()`将索引0处添加空间，然后将值存储到这个地方，并将列表中既有的每个元素都右移一个位置。
### 1.3.从列表中删除元素
#### 1.3.1.使用del语句删除元素
```python
del motorcycles[0]
```
#### 1.3.2.使用pop()删除元素
删除列表末尾的元素，并可以接着使用它
```python
popped_motorcycle = motorcycles.pop()
```
#### 1.3.3.使用pop()弹出列表中任何位置的元素
```python
popped_motorcycle = motorcycles.pop(1)
```
#### 1.3.4.根据值删除元素
使用remove()
```python
motorcycles.remove('ducati')
```
- remove()只能删除第一个指定的值。
## 2.组织列表
### 2.1.使用sort()对列表进行永久排序
默认从小到大
```python
cars = ['bmw','audi','toyota','subaru']
cars.sort()
print(cars)
```
output:`['audi','bmw','subaru','subaru']`

**反转**:使之从大到小
```python
cars.sort(reverse=True)
```
### 2.2.使用sorted()对列表临时排序
```python
print(sorted(cars))
```
### 2.3.倒着打印列表
```python
cars.reverse()
```
### 2.4.确定列表的长度
```python
len(cars)
```
Output:`4`
