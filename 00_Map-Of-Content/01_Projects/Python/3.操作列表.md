Python根据缩进来判断代码行与前一个代码行的关系。
```python
for magician in magicians:
	print(magician)
```
- 注意缩进
- 注意冒号，不要缺少冒号
## 1.创建数值列表：
### 1.1.使用函数range()
- range()可以生成一系列数
- 让Python从指定的第一个值开始数，并在到达指定的第二个值停止。然而**输出并不包含第二个值**。
### 1.2.使用range()创建数字列表：
- list()将range()的结果直接转换为列表：
```python
numbers = list(range(1,6))
print(numbers)
```
- 使用range()时，还可以给定第三个参数来指定步长。
```python
even_numbers = list(range(2,11,2))
print(even_numbers)
```
### 1.3.对数字列表执行简单的统计计算：
```python
digits = [1,2,3,4,5,6,7,8,9,0]
min(digits)
max(digits)
sum(digits)
```
### 1.4.列表解析：
将for循环和创建新元素的代码合并成一行，并自动附加新元素。
```python
squares = [value**2 for value in range(1,11)]
print(squares)
```

## 3.元组
不可变的列表称为**元组**。
### 3.1.定义元组
元组是由逗号标识的，圆括号只是让元组看起来更整洁、更清晰。如果要定义只包含一个元素的元组，必须在这个元素后面加上括号：
`my_t = (3,)`
### 3.2.修改元组变量
如果要进行修改，可以重新定义整个元组
```python
dimensions = (200,50)

dimensions = (400,100)
```
