## Description

给你一个二维整数数组 `matrix`， 返回 `matrix` 的 **转置矩阵** 。
矩阵的 **转置** 是指将矩阵的主对角线翻转，交换矩阵的行索引与列索引。![[Pasted image 20251107092101.png]]
### Example:  
Input: 
```json
[[1,2,3],[4,5,6],[7,8,9]]
```

Output: 
```json
[[1,4,7],[2,5,8],[3,6,9]]
```
## Question Link

[LeetCode](https://leetcode.cn/problems/transpose-matrix/)
## Approach

- 首先创建一个相反的列表存储矩阵
- 然后进行转置操作


## Implementation

```python
class Solution:
    def transpose(self, matrix: List[List[int]]) -> List[List[int]]:
        m, n = len(matrix), len(matrix[0])       # 原矩阵行数和列数
        res = [[0] * m for _ in range(n)]        # 创建 n 行 m 列的空矩阵
        for i in range(m):
            for j in range(n):
                res[j][i] = matrix[i][j]         # 转置操作
        return res
```

## Analysis
` res = [[0] * m for _ in range(n)]        # 创建 n 行 m 列的空矩阵`
	- 关键是这一句，因为**Python无法直接像C那样`res[m][n]`**，所以需要借助列表操作
	- `[0] * m`意味着创建一个m个0元素的列表，这是一个列表。
	- `for _ in range(n)`代表循环n次，即有n个有着m个0元素的列表的列表。
