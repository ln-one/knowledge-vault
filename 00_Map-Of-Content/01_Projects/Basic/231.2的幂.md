## Description

给你一个整数 `n`，请你判断该整数是否是 2 的幂次方。如果是，返回 `true` ；否则，返回 `false`。
如果存在一个整数 `x` 使得 `n == 2x` ，则认为 `n` 是 2 的幂次方。
### Example:  
Input: 
```json
n = 1
```

Output: 
```json
true
```
## Question Link

[LeetCode](https://leetcode.cn/problems/power-of-two)
## Approach

- 运用位运算的方法来解决问题，实现常数的算法复杂度。

## Implementation

```python
class Solution:
    def isPowerOfTwo(self, n: int) -> bool:
        return n > 0 and (n & (n - 1)) == 0
```

## Analysis
- 关键点在于：如果一个数是2的幂，那么它的位模式则应该为：**1000 0000**类似的，那么此时该数减去1:**0111 1111**。此时两者相与：一定得**0000 0000**，即为0。
