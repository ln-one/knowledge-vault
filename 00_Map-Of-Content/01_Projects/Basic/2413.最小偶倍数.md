## Description
给你一个正整数 `n` ，返回 `2` 和 `n` 的最小公倍数（正整数）。
### Example:  
Input: 
```json
n = 5
```

Output: 
```json
10
```
## Question Link

[LeetCode](https://leetcode.cn/problems/smallest-even-multiple)
## Approach

- **最小公倍数一定是比两个数都要大的或相等的**。
- 如果数N可以被2整除，说明N本身是最小公倍数。
- 否则最小公倍数应该是2\*n
## Implementation

```python
class Solution:
def smallestEvenMultiple(self, n: int) -> int:
	if n % 2 == 0:
		return n
	else :
		return(n * 2)
```
