## Description

给定一个整数，写一个函数来判断它是否是 3 的幂次方。如果是，返回 `true` ；否则，返回 `false` 。

整数 `n` 是 3 的幂次方需满足：存在整数 `x` 使得 $n == 3^x$
### Example:  
Input: 
```json
n = 27
```

Output: 
```json
true
```
## Question Link

[LeetCode](https://leetcode.cn/problems/power-of-three)
## Approach

- 首先，如果是小于等于0，则说明一定是false;
- 如果不是小于或等于0，那么思考3的幂次方数的特点：
	1. 首先该数一定可以被3整除或者是1。
	2. 再次思考，它一直被3除，直到它无法被3整除，此时剩下的数只能是1。


## Implementation

```python
class Solution:
    def isPowerOfThree(self, n: int) -> bool:
        if n <= 0:
            return False
        while n % 3 == 0:
            n //= 3
        return n == 1
```
