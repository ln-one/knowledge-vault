## Description

将摄氏度转变成kervin 和 fahrenheit
### Example:  
Input: 
```json
celsius = 36.50
```

Output: 
```json
[309.65000,97.70000]
```
## Question Link

[LeetCode - 温度转换](https://leetcode.cn/problems/convert-the-temperature)
## Approach

- 非常简单，只不过一开始不太熟悉力扣的刷题过程。

## Implementation

```python
class Solution:

    def convertTemperature(self, celsius: float) -> List[float]:

        kervin = celsius + 273.15

        fahrenheit = celsius * 1.80 + 32.00

        return [kervin,fahrenheit]
```

## Analysis
- 返回一个列表List，那么就加`[]`。
