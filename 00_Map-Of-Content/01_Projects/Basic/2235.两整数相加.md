## Description
给你两个整数 `num1` 和 `num2`，返回这两个整数的和。
### Example:  
Input: 
```json
num1 = 12, num2 = 5
```

Output: 
```json
17
```
## Question Link

[LeetCode - Two Sum](https://leetcode.cn/problems/add-two-integers)
## Approach

- 直接返回两个数字相加的结果即可

## Implementation

```python
class Solution:
    def sum(self, num1: int, num2: int) -> int:
        return num1+num2
```
